Why learn Javascript?

HTML and CSS design the core and base of every website needed to make it functional and viewable by the public. However, what sets good websites apart from great is the utilization of Javascript.

Javascript is a programming language often used by websites to bring animations or action to pages that HTML or CSS cannot. It can be imbedded either internally or externally via script. Javascript helps accomplish many daunting tasks on the OpenWeb page such as the placeholder animation at the front of our site and fading in our smaller sticky header you see right now.

Javascript itself is a very flexible language with multiple ways to accomplish one task. jQuery, a library of Javascript, can help make the syntax even easier. Learn Javascript with OpenWeb.
